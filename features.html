Lupyd uses Lib Singal - Protocal for chats Encryption

libsignal is the official open-source cryptographic library created by Signal.
It contains all the low-level code needed to implement the Signal Protocol inside any messaging app.
You can think of libsignal as a toolkit that developers use to add end-to-end encryption (E2EE) without writing complex cryptography from scratch.

libsignal uses prekeys for asynchronous messages You can send messages even when the other user is offline
The server stores temporary prekeys
The sender uses them to secretly set up encryption
This is how apps like lupyd and Signal allow smooth messaging without both users being online together.
